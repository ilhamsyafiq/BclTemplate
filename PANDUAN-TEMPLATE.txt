================================================================================
                           TIPS PEMILIHAN GAMBAR
================================================================================

1. SAIZ GAMBAR DISYORKAN:
   - Hero/Banner: 1200x800px atau 16:9 ratio
   - Produk: 800x800px (square) atau 4:3 ratio
   - Thumbnail: 400x400px
   - Profile/Avatar: 200x200px

2. FORMAT:
   - JPG untuk foto
   - PNG untuk gambar dengan transparency
   - WebP untuk fail lebih kecil (jika browser support)

3. HOSTING GAMBAR PERCUMA:
   - Imgur.com
   - ImgBB.com
   - Cloudinary.com
   - Unsplash.com (stok gambar percuma)

4. OPTIMASI:
   - Compress gambar sebelum upload
   - Guna lazy loading untuk gambar banyak
   - Pastikan URL gambar adalah HTTPS


================================================================================
                              CARA MENGEDIT
================================================================================

1. Buka fail HTML dengan text editor (VS Code, Notepad++, dll)

2. Cari bahagian "const CONFIG = {" di bahagian atas fail

3. Edit nilai-nilai dalam CONFIG mengikut keperluan

4. JANGAN edit kod di bawah garisan:
   "DO NOT EDIT BELOW THIS LINE"

5. Simpan fail dan refresh browser untuk lihat perubahan

6. Untuk payment form, tambah kod form anda di bahagian:
